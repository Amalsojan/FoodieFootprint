FoodieFootprint is a Chrome extension that helps you track and visualize your food spending trends on **Zomato** and **Swiggy**.

## Features
- **Dual Platform Support**: Sync data from both Zomato and Swiggy.
- **Visual Analytics**: Interactive charts showing monthly spending, top restaurants, and time-of-day trends.
- **Smart Detection**: Automatically detects the active platform tab for easy syncing.
- **IST Time Tracking**: Displays the last sync time in Indian Standard Time (IST).
- **Premium UI**: Modern, clean design with dynamic themes (Red for Zomato, Orange for Swiggy).
- **Progress Tracking**: Real-time progress bar during data synchronization.

## Installation
1. Download the repository as a ZIP or clone it.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the extension folder.

## Usage
1. Open the extension from your browser toolbar.
2. Select the platform (Zomato or Swiggy).
3. Click **Sync Data**. If you're not logged in, the extension will prompt you to open the site and log in.
4. view your food footprint!
